# Monsters-Clicker
A simple clicker that challenges you to use all your strength to hit the mouse and defeat the monster that is out of control.

![image](https://github.com/user-attachments/assets/9b8f182f-9947-4fce-ab8e-d9eb52240f66)

This game was made for ‘TODO Game Jam’. The time period to make the game was 7 days and the theme was ‘Out of Control’. The game is a simple clicker with 2 levels. One easy and one hard. Your mission is to defeat the monsters before they destroy the city, for that you will need the help of the police and the army.
